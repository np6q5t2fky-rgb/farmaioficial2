# Запуск на Телефоні - Покрокова Інструкція

## Підготовка (на комп'ютері)

### Крок 1: Дізнайтеся IP адресу вашого комп'ютера

**Windows PowerShell:**
```powershell
ipconfig
```
Знайдіть рядок "IPv4 Address" - це буде щось типу `192.168.0.XXX` або `192.168.1.XXX`

Наприклад: `192.168.0.251`

### Крок 2: Запустіть програму

**Варіант A: Автоматичний запуск (рекомендовано)**
```powershell
cd d:\FARMAI
.\start-all.bat
```

**Варіант B: Ручний запуск**

Термінал 1 (Backend):
```powershell
cd d:\FARMAI\backend
python app.py
```

Термінал 2 (Frontend):
```powershell
cd d:\FARMAI\frontend
npm start
```

### Крок 3: Перевірте що все працює

Backend має показати:
```
Сервер запущено на http://localhost:5000
 * Running on http://192.168.0.251:5000
Gemini API: Налаштовано
```

Frontend має показати:
```
Compiled successfully!
Local:            http://localhost:3000
On Your Network:  http://192.168.0.251:3000
```

### Крок 4: Вимкніть фаєрвол Windows (тимчасово для тесту)

1. Натисніть Win + R
2. Введіть: `firewall.cpl`
3. Зліва: "Turn Windows Defender Firewall on or off"
4. Виберіть "Turn off" для приватної мережі
5. OK

**АБО дозвольте порти:**
```powershell
# Запустіть PowerShell як адміністратор
New-NetFirewallRule -DisplayName "FarmAI Backend" -Direction Inbound -LocalPort 5000 -Protocol TCP -Action Allow
New-NetFirewallRule -DisplayName "FarmAI Frontend" -Direction Inbound -LocalPort 3000 -Protocol TCP -Action Allow
```

## На Телефоні

### Крок 1: Підключіться до тієї ж WiFi мережі

Телефон та комп'ютер МАЮТЬ бути в одній WiFi мережі!

### Крок 2: Відкрийте браузер на телефоні

Рекомендовано: Chrome або Safari

### Крок 3: Введіть адресу

Замініть `192.168.0.251` на IP вашого комп'ютера з Кроку 1:

**Для розробки (2 сервери):**
```
http://192.168.0.251:3000
```

**Для production збірки (1 сервер):**
```
http://192.168.0.251:5000
```

### Крок 4: Увійдіть

Логін: `admin`
Пароль: `admin`

## Налаштування для зручності

### Додати на головний екран (iOS)

1. Відкрийте сайт в Safari
2. Натисніть кнопку "Поділитися" (квадрат зі стрілкою)
3. Прокрутіть вниз → "Add to Home Screen"
4. Назвіть "Свиноферма" → Add

### Додати на головний екран (Android)

1. Відкрийте сайт в Chrome
2. Меню (три крапки) → "Add to Home screen"
3. Назвіть "Свиноферма" → Add

Тепер у вас буде іконка як додаток!

## Вирішення Проблем

### "Сайт недоступний"

**Перевірте:**
1. Чи комп'ютер та телефон в одній WiFi?
2. Чи працює backend? (перевірте термінал)
3. Чи правильний IP? (запустіть `ipconfig` знову)
4. Чи вимкнений фаєрвол?

**Тест:** На комп'ютері відкрийте в браузері:
```
http://192.168.0.251:5000/api/health
```
Має показати: `{"status":"ok"}`

### Повільно завантажується

- Перезапустіть WiFi роутер
- Переконайтеся що WiFi 5GHz (швидший)
- Закрийте інші програми на комп'ютері

### Таблиця не прокручується

- Використайте два пальці для прокрутки
- Спробуйте горизонтально свайпнути по таблиці

### Текст дуже дрібний

- Збільште шрифт в налаштуваннях телефону
- Або зумуйте браузер (два пальці)

## Production Mode (для постійного використання)

Якщо хочете щоб працювало тільки на одному порті:

### Крок 1: Зберіть frontend
```powershell
cd d:\FARMAI\frontend
npm run build
```

### Крок 2: Скопіюйте в backend
```powershell
xcopy /E /I /Y build ..\backend\frontend_build
```

### Крок 3: Запустіть тільки backend
```powershell
cd d:\FARMAI\backend
python app.py
```

### Крок 4: Відкрийте на телефоні
```
http://192.168.0.251:5000
```

Тепер все працює на одному порті!

## Поради

1. **Тримайте комп'ютер увімкненим** поки працюєте з телефону
2. **Не закривайте термінали** де запущені сервери
3. **IP адреса може змінитись** після перезавантаження роутера
4. **Для постійного доступу** краще використати VPS (див. VPS_DEPLOYMENT.md)
