# Мобільний Додаток через Expo Go

## Що таке Expo Go?

Expo Go - це додаток для iOS/Android, який дозволяє запускати React Native програми без складного налаштування. Просто скануєте QR код і все працює!

## Переваги:
✅ Не потрібно будувати APK/IPA
✅ Миттєві оновлення (hot reload)
✅ Працює на iOS та Android
✅ Простіше ніж веб-версія

---

## Встановлення

### Крок 1: Встановіть Expo CLI на комп'ютері

```powershell
npm install -g expo-cli
```

### Крок 2: Створіть мобільний проект

```powershell
cd d:\FARMAI
npx create-expo-app mobile-app --template blank
```
