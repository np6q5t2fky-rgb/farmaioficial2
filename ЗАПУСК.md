# Як запустити мобільний додаток "Облік Свиноферми"

## 1. Запуск на комп'ютері

### Backend (Flask сервер)
```powershell
cd d:\FARMAI\backend
python app.py
```
Сервер запуститься на http://192.168.0.251:5000

### Expo Metro (для мобільного додатку)
```powershell
cd d:\FARMAI\mobile
npm start
```
З'явиться QR код для підключення

## 2. Підключення телефону

1. **Встанови Expo Go на телефон:**
   - Android: Google Play Store → "Expo Go"
   - iOS: App Store → "Expo Go"

2. **Відскануй QR код:**
   - Відкрий Expo Go на телефоні
   - Натисни "Scan QR code"
   - Наведи камеру на QR код в терміналі

3. **Готово!** Додаток завантажиться на телефоні

## 3. Використання додатку

- **Меню:** Натисни ☰ зверху зліва
- **Вкладки:**
  - Консультант - чат з AI про свинарство
  - База знань - додавання/видалення записів
  - Облік - перегляд та редагування таблиць Excel

## Важливо

- Backend має бути запущений для роботи додатку
- Телефон і комп'ютер мають бути в одній WiFi мережі (або використовуй tunnel)
- Excel файл знаходиться: `d:\FARMAI\тижневий облік.xlsx`

## IP адреса

Якщо треба змінити IP:
1. Відкрий `d:\FARMAI\mobile\config.js`
2. Зміни `API_URL` на твій IP
