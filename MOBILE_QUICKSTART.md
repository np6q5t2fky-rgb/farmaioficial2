# ШВИДКИЙ СТАРТ НА ТЕЛЕФОНІ

## Крок 1: На комп'ютері

### Запустіть програму:
```powershell
cd d:\FARMAI
.\start-mobile.bat
```

Скрипт покаже вашу IP адресу, наприклад:
```
IP адреса комп'ютера: 192.168.0.251

На телефоні відкрийте:
  http://192.168.0.251:3000
```

## Крок 2: Вимкніть фаєрвол (якщо потрібно)

Win + R → `firewall.cpl` → Turn off для приватної мережі

**АБО** дозвольте порти в PowerShell (як адміністратор):
```powershell
New-NetFirewallRule -DisplayName "Свиноферма Backend" -Direction Inbound -LocalPort 5000 -Protocol TCP -Action Allow
New-NetFirewallRule -DisplayName "Свиноферма Frontend" -Direction Inbound -LocalPort 3000 -Protocol TCP -Action Allow
```

## Крок 3: На телефоні

1. **Підключіться до тієї ж WiFi** що і комп'ютер
2. **Відкрийте браузер** (Chrome або Safari)
3. **Введіть адресу** яку показав скрипт (наприклад: `http://192.168.0.251:3000`)
4. **Увійдіть**: admin / admin

## Додати іконку на головний екран

**iOS (iPhone/iPad):**
Safari → Поділитися → Add to Home Screen → "Свиноферма"

**Android:**
Chrome → Меню (⋮) → Add to Home screen → "Свиноферма"

## Проблеми?

**Не відкривається:**
- Перевірте WiFi (комп'ютер і телефон в одній мережі)
- Перевірте IP адресу (`ipconfig` на комп'ютері)
- Вимкніть фаєрвол

**Таблиця не прокручується:**
- Свайпайте двома пальцями
- Прокручуйте горизонтально по таблиці

**Повільно:**
- Закрийте інші програми на комп'ютері
- Використайте WiFi 5GHz
